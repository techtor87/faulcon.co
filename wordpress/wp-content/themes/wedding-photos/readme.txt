== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in Wedding Photos in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/wedding-photos/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (wedding-photos.zip) that you have downloaded. Via FTP, upload the whole folder (wedding-photos) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(wedding-photos). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/wedding-photos/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Wedding Photos theme to activate it.
	- Want to enjoy Wedding Photos feature. Go to  Dashboard -> Appearance -> Customize and play with Wedding Photos Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.

	Note: Parent Theme Photograph must be installed to work with Wedding Photos Child theme
